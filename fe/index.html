How to Use the Plane Ticket Analyzer

Follow these steps to get started with the app:

  1. Start the Web Scraper:

    2. Open a terminal and navigate to the be folder. Then, type: python ws.py
    Note: Ensure Python 3 is installed on your system.

  2. Start the Analysis Tool:

    3. Open another terminal and navigate to the be folder. Then, type: python ml.py
    Note: Ensure Python 3 is installed on your system.

  3. Input Guidelines:

    4. - The data input is not case sensitive, so you can type locations in any capitalization.
    - Use the format: City-Country (e.g., paris-france).

  4. Date Selection:

    5. - Ensure the selected dates are valid.
    - Dates in the past will result in an error.

If you encounter any issues, double-check the instructions above or contact support for assistance.
